## Question

### Editable Comment List

**Scenario**

Users can view and edit comments on a blog.

**Requirements**

- Load exactly 3 comments initially.
- Click comment to enter edit mode.
- Show input + Save + Cancel.
- Save applies changes.
- Cancel discards changes.
- Only one comment editable at a time.
- Switching edit discards previous edits.

**Bonus**

- Keyboard save (Enter)
- Escape cancels edit
- Auto-focus input

**Note**

> I may not have met all points as I tried to do it in a time constraint similar to the interview
